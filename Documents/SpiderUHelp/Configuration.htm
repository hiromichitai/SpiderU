Configuration menu


There are four tabs for configuration.

  • File format
  • File name
  • CSV option
  • Synchronize

File format

File format is selected by choosing from combobox.Supported file formats are as below:

  • CSV(comma separated value) file

    Comma separated text file. Character code is UTF-8.

  • H5(HDF5) file

    HDF5(Hierachical Data File 5) is a portable and extensible file format. SpiderU writes captured data as array of double with attributes. Detailed information about HDF5 will be found in the HDF5 home page at HDF5 home page.

  • LD1 file

    LD1 is a locally defined binary format.LD1 has one header and one or more data groups. Each data groups has two or more data sets,and each data set has one label string and data array of single float. The length of data array is defined for each data group.

File name

SpiderU can supply automatic file name as default output file name.In configuration menu, you can set startup meta-string for file name prefix, effective file name prefix, suffix, serial number and number of digits of serial number.
