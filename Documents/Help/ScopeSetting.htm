Setting DSOs

ScopeSetting form provides configuration of each DSO.

  • Comment: Comma string for each DSO
  • Label: Label string for each channel of DSO
  • Unit: Unit string for each channel of DSO
  • Multiplier: Multiplying factor for each channel data

Caution: Although this form accepts multi-byte character, whether output file contains correct string or not depends on the format of output file.
