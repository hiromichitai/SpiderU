Connecting DSO

Three ways of connection is supported:

 Bus  PC hardware requirements  Required device driver
 GP-IB(IEEE 488)  National Instruments GP-IB controllers  National Instruments IEEE488.2 driver
 USB TMC  No additional hardware required  Yokogawa T&M device driver
 USB PHY   No additional hardware required  libusb

Requirements for Installation of NI device driver

In order to utilize National Instruments software (device driver and application software interface ), you must fullfill following requirements:

  • Device driver version should be 14.0 or later
